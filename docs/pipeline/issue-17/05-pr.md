2026年のZ世代向けトレンド（Liquid Glass・大胆タイポ・鮮やか色アクセント・柔影・軽モーション）を参考に、**「色」を主役にした UI/UX リフレッシュ**。最も波及効果が高い**デザイントークン＋共有コンポーネント**を刷新し、全画面が新しい見た目を継承する。地（背景）はモノクロ維持で過剰刺激を避け、彩度は12色（COLOR_POOL）と単一 tint に集約。
