## 堅牢性（レビュー反映済み）
- 逐次直列化（mutationChain）＋ `kick()` 自己再キックで**処理中 enqueue の取りこぼしを防止**。
- `uploading→pending` の rehydrate 正規化で再起動中の固着を解消。永続化 write 失敗はメモリ維持＋握り潰し。壊れ JSON は空配列で開始。
