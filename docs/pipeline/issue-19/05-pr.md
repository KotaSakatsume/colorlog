## 含まれるもの
- `src/repositories/firebase/`（新規・native 隔離）:
  - `firebase-auth-service.ts`: 匿名サインイン開始、`linkWithApple()`（expo-crypto で raw nonce + SHA-256→Apple、idToken+rawNonce→`AppleAuthProvider.credential`→`linkWithCredential`）。
  - `firebase-trip-repository.ts`: `onSnapshot` 購読、`runTransaction`（`tx.get`）で `assignColors`＝既存純関数 `assignColorsToTrip` 再利用、`joinTrip`＝`pickColorForJoiner` 再利用、create/delete/inviteCodes。
  - `firebase-post-repository.ts`: posts `onSnapshot`（orderBy+limit50）、`promotePhoto`＝決定的 postId `${uid}_${slotIndex}`+`tx.get` で postCount≤9・slot差し替え（**画像→Storageは注入 `PhotoUploader` passthrough スタブ**＝§9-7）、caption 200字検証、reactions（`reactions/{uid}`+非正規化 `reactionCounts` の `increment`）。
  - `adapters.ts`: Timestamp⇄Date。**`createdAt`/`lastPostAt`=`serverTimestamp()`**（本番ルール `serverTimestamped()` 整合）、`startDate`/`endDate`/`expiresAt`=`Timestamp.fromDate`。
  - `photo-uploader.ts` / `firebase-app.ts` / `index.ts`(`createFirebaseRepositories`)。
- `context.tsx`: `Platform.OS!=='web' && !isExpoGo && FIREBASE_ENABLED` ガード内の**動的 require + try**。既定 Mock。
- **`firestore.rules`**: reactions サブコレクション（read=メンバー / write=自uid+確定絵文字）と posts の `reactionCounts` 限定 update を追加。`tests/rules` に許可/拒否テスト追加。
- `package.json`: `expo-crypto` 追加（Apple nonce）。
